### Assignment1
_Assignment1_data.csv_ contains data from **5** popular books. It states the _title_, the _author_, the _year_ of publication, and the _genre_ of the books.  
The source of the data is [Google Books](https://books.google.com). _Assignment1_metadata.csv_ contains the structure and a description of the raw data files (the book covers).  
The structure of the data csv file is shown in the table underneath.   

| Title | Author | Year | Genre |
|-------|--------|------|----------|
|The title of the book | The author of the book | The year the book was published | The genre Google Books places the books in |
